Passo Inicial
Crie uma Pasta para o Projeto Inicial
Por Exemplo     - Personal Budget
Add Este no Git - git init
			git add .
			git commit -m "First Commit"
			git remote add origin https://github.com/Douglas-Bonelli/PersonalBudget.git
			git remote -v
			git push origin master

Neste Ponto vc já tem no GIT o seu Projeto.
Dentro deste Projeto serão incluidos todos os services pertinentes deste projeto.

No Nosso Caso o Service User
		     Dentro da Pasta Personal Budget
		     Inclua a Pasta user-service
		     Inclua um Arquivo txt = Documentação de Criação do User Service

			git add .
			git commit -m "Creating user-service"
			git push origin master


A partir deste Ponto Sua Estrutura Principa do Projeto esta Pronta.
Dentro da Pasta de user-service existe a documentação para criação de um new service.

---------------------------------------------------------------------------------------------

Sempre que precisar incluir um novo service siga estes passos:
	Dentro da Pasta Principa do Projeto = Personal Budget
		     Inclua a Pasta           = user-service
		     Inclua um Arquivo txt    = Documentação de Criação do User Service

			git add .
			git commit -m "Creating user-service"
			git push origin master
